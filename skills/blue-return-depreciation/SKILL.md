# 減価償却資産の入力値算出

指定された年度の固定資産データから、国税庁 確定申告作成コーナー「減価償却資産の入力」画面の入力値を算出して Markdown 形式で出力する。

## 前提

- 対象年度の固定資産台帳データ（取得日、取得価額、耐用年数、償却方法等）が利用可能であること
- データソースから必要な情報が得られない場合はユーザーに確認すること

## 出力項目

### 各資産の明細

保有する減価償却資産ごとに以下を出力する:

| 項目 | 説明 |
|------|------|
| 資産の名称 | 資産の名前（例: MacBook Pro 14インチ） |
| 取得年月 | 取得した年月 |
| 取得価額 | 購入時の価額 |
| 償却方法 | 定額法 / 定率法 |
| 耐用年数 | 法定耐用年数 |
| 本年分の償却費 | 当該年度の減価償却費（初年度は月割計算） |
| 期末帳簿価額 | 取得価額 − 累計償却額 |
| 未償却残高 | 期末帳簿価額と同値 |

### 合計

- **本年分の減価償却費の合計額**（blue-return-income-statement の減価償却費 #18 と一致すること）

## 出力形式

Markdown テーブル。資産ごとに1ブロック。最後に合計額。

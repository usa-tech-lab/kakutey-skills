# 貸借対照表（一般用）の入力値算出

指定された年度の会計データから、国税庁 確定申告作成コーナー「貸借対照表（一般用）の入力」画面の入力値を算出して Markdown 形式で出力する。

## 前提

- 対象年度の貸借対照表データ（期首・期末の各勘定残高）が利用可能であること
- データソースから必要な情報が得られない場合はユーザーに確認すること

## 重要な注意事項

- **事業主貸・事業主借**: 決算処理（closing）前の値を使用すること。決算後のデータしかない場合は、仕訳データから事業主貸（debit合計）・事業主借（credit合計）を別途算出する
- **元入金**: 期首の値を期末にもそのまま使用する（年中に変動しない）
- **事業主貸は期末のみ**（資産の部に表示）、**事業主借は期末のみ**（負債・資本の部に表示）

## 出力項目

### 期間

- 期首: 1月1日
- 期末: 12月31日

### 資産の部

各科目の期首・期末の金額を出力する（該当なしは 0）:

| 科目 |
|------|
| 現金 |
| 当座預金 |
| 定期預金 |
| その他の預金 |
| 受取手形 |
| 売掛金 |
| 有価証券 |
| 棚卸資産 |
| 前払金 |
| 貸付金 |
| 建物 |
| 建物附属設備 |
| 機械装置 |
| 車両運搬具 |
| 工具・器具・備品 |
| 土地 |
| 事業主貸（期末のみ） |

### 負債・資本の部

各科目の期首・期末の金額を出力する（該当なしは 0）:

| 科目 |
|------|
| 支払手形 |
| 買掛金 |
| 借入金 |
| 未払金 |
| 前受金 |
| 預り金 |
| 貸倒引当金 |
| 事業主借（期末のみ） |
| 元入金 |

### 検算

- 期首: 資産の部 合計 = 負債・資本の部 合計
- 期末: 資産の部 合計 = 負債・資本の部 合計（青色申告特別控除前の所得金額を含む）

## 出力形式

Markdown テーブル。科目 × 期首/期末 の一覧。最後に検算結果。

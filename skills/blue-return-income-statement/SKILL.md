# 決算書（一般用）の入力値算出

指定された年度の会計データから、国税庁 確定申告作成コーナー「決算書（一般用）の入力」画面の入力値を算出して Markdown 形式で出力する。

## 前提

- 対象年度の会計データ（損益計算書相当）が利用可能であること
- データソースから必要な情報が得られない場合はユーザーに確認すること

## 出力項目

### 期間

- 自: 1月1日
- 至: 12月31日

### 売上（収入）金額

- **(1) 売上（収入）金額**: 年間の売上合計

### 売上原価

- **(2) 期首商品棚卸高**
- **(3) 仕入金額**
- **(5) 期末商品棚卸高**

### 経費（標準科目 #8〜#24）

以下の全17科目について金額を出力する（該当なしの科目は 0 と出力）。

| # | 科目 |
|---|------|
| 8 | 租税公課 |
| 9 | 荷造運賃 |
| 10 | 水道光熱費 |
| 11 | 旅費交通費 |
| 12 | 通信費 |
| 13 | 広告宣伝費 |
| 14 | 接待交際費 |
| 15 | 損害保険料 |
| 16 | 修繕費 |
| 17 | 消耗品費 |
| 18 | 減価償却費 |
| 19 | 福利厚生費 |
| 20 | 給料賃金 |
| 21 | 外注工賃 |
| 22 | 利子割引料 |
| 23 | 地代家賃 |
| 24 | 貸倒金 |

### 経費（任意科目 #25〜#30）

上記の標準科目に該当しない経費がある場合、科目名と金額を出力する。最大6科目まで。

### 自動計算される項目（参考）

- **経費合計**: 標準科目＋任意科目の合計
- **差引金額（所得）**: 売上 − 売上原価 − 経費合計

## 出力形式

Markdown。テーブル形式で科目・金額を一覧化すること。

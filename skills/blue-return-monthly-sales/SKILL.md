# 売上（収入）金額・仕入金額の入力値算出

指定された年度の会計データから、国税庁 確定申告作成コーナー「売上（収入）金額・仕入金額の入力」画面の入力値を算出して Markdown 形式で出力する。

## 前提

- 対象年度の売上・仕入に関する仕訳データ（日付・金額・摘要）が利用可能であること
- データソースから月別の内訳が得られない場合はユーザーに確認すること

## 出力項目

### 月別売上（収入）金額・仕入金額

1月〜12月の各月について以下を出力する:

| 項目 | 説明 |
|------|------|
| 月 | 1月〜12月 |
| 売上金額 | その月の売上合計（0の月も出力） |
| 仕入金額 | その月の仕入合計（0の月も出力） |
| 備考 | 売上先名・案件名など |

### 合計

- 売上金額の年間合計
- 仕入金額の年間合計

## 出力形式

Markdown テーブル。12行（1月〜12月）＋合計行。

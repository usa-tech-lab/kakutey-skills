# 消費税 決算額・課税取引金額の内訳等の入力値算出

指定された年度の会計データから、国税庁 確定申告作成コーナー「決算額・課税取引金額の内訳等の入力（事業所得（営業等））」画面の入力値を算出して Markdown 形式で出力する。

## 前提

- 対象年度の損益計算書データ（売上・経費の各科目金額）が利用可能であること
- 消費税区分の判定には [tax_classification.json](references/tax_classification.json) を参照すること
- 消費税区分の判断が難しい経費はユーザーに確認すること
- データソースから必要な情報が得られない場合はユーザーに確認すること

## 列の定義

| 列 | 説明 |
|----|------|
| A 決算額 | 決算書の税込金額 |
| B 課税取引にならないもの | 不課税・非課税の金額 |
| C 課税取引金額 | A − B（自動計算） |
| D+E 軽減税率（8%）適用分 | 飲食料品等の軽減税率対象 |
| E 免税事業者等取引分（軽減） | インボイス未登録事業者からの仕入（軽減税率分） |
| F+G 標準税率（10%）適用分 | 通常の課税取引 |
| G 免税事業者等取引分（標準） | インボイス未登録事業者からの仕入（標準税率分） |

## 出力項目

### (1) 売上（収入）金額

A, B, D+E, F+G の各列の金額

### (2)〜(6) 売上原価

期首棚卸・仕入・期末棚卸の各列の金額

### 経費科目ごとの消費税区分

blue-return-income-statement で出力した各経費科目（#8〜#24、任意科目）について、A, B, D+E, F+G の各列の金額と消費税区分の根拠を出力する。

### 経費計

各列（A, B, D+E, F+G）の合計

### 検算

- A = B + C
- C = D+E + F+G

### 質問項目

以下の各項目に「はい」or「いいえ」で回答:

| 質問 |
|------|
| 発生した貸倒金の金額の入力 |
| 回収した貸倒金の金額の入力 |
| 保税地域からの引き取り貨物に係る金額の入力 |
| 課税仕入れに係る対価の返還等の金額の入力 |
| 令和N年1月1日から課税事業者となった方の棚卸高の調整の入力 |
| 令和N+1年分に免税事業者となる方の棚卸高の調整の入力 |

## 消費税区分の判定ルール

[tax_classification.json](references/tax_classification.json) に従い分類する:

- **non_taxable**: 全額を B 列に計上する科目（減価償却費、租税公課、損害保険料、給料賃金、利子割引料、貸倒金 等）
- **non_taxable_custom**: 全額を B 列に計上するが、事業固有の理由がある科目
- **reduced_8**: D+E 列に計上する科目（飲食料品の仕入等）
- **上記以外**: F+G 列に計上（標準税率 10%）

## 出力形式

Markdown テーブル。科目ごとに A/B/D+E/F+G の各列と消費税区分の根拠を一覧化。
